Git 安装

  • 下载 Git
    我们可以去官网下载 地址：https://git-scm.com/downloads（由于国内下载较慢）
    这里提供一个国内的下载网站 地址：https://github.com/waylau/git-for-win

  • 安装 Git

    安装Git完成，可以回到我们桌面，右键就可以看到Git Bash here和Git GUI here

发布网站

在我们的blog根目录里面 有一个文件叫_config.yml ，这个文件被称为站点配置

进入themes中，还有一个相同的文件，这个文件被叫做主题配置

  • 关联 Github
    打开站点配置文件_config.yml拉到最后修改
    deploy:
    type: git
    repo: 这里填入GitHub仓库的完整路径，记得加上 .git
    branch: master
    三个冒号后面都有一个空格

    保存文件，然后我们回到blog目录，安装Git部署文件

  • 部署 Hexo

  • 测试

绑定域名

这里我们要先购买域名，各大网站都会有域名出售，建议去阿里云购买

  • 解析 域名
    我们先找到我们的域名，点进去解析

    在这里我们要添加三条解析记录
    192.30.252.153是GitHub的地址
    第三个记录类型是CNAME，CNAME的记录值是：你的用户名.github.io

  • 设置 域名
    我们返回我们的Github仓库，继续是进入settings，找到Github Pages
    设置Custom domain，输入你的域名

    保存后，我们找到文件夹，进入blog/source目录，创建一个文件输入自己的域名
    保存退出，回到我们的blog目录
    打开我们的命令行，依次输入之前我们重新部署的命令

  • 测试 域名
    我们打开浏览器，输入你刚刚自己的个性化域名，就会直接进入你的个人博客

    这是我的个人博客，欢迎进行观看 博客：http://www.hewenhao.top/
