Servlet 是运行在 Web 服务器端的 java 应用程序，可以生成动态的 Web 页面

3.Servlet生命周期

1.init()方法

第一次调用Servlet时，将会创建一个实例

2.doGet()/doPost()/service()方法

doGet()以get方式请求Servlet时运行。常见的get请求方式链接，get方式表单提交，直接访问等。
doPost()以post方式请求Servlet时运行。常见的post请求方式为post方式表单提交。

3.destory()方法

destory()方法在Servlet实例消亡的时候自动调用。

4.Servlet与JSP内置对象

1.获取out对象

JSP中的out对象对应于Servlet中”javax.servlet.jsp.JspWriter”.

2.获得request和response对象

Servlet获取JSP中的request和response很容易，因为参数都传给了doXXX()方法中了

3.获得session对象

session对象对应的是HttpSession接口

4.获得application对象

application对象对应的是ServletContext接口

除此之外，还可以使用application来实现服务器内跳转
(1)重定向

(2)服务器内跳转
