本文字数： 9.2k 阅读时长 ≈ 8 分钟

1.认识JavaBean

实际上，JavaBean就是一种Java的组件控件
JavaBean支持两组控件：可视化组件和非可视化组件。对于可视化组件，开发人员可以在运行的结果中
看到界面效果；而非可视化组件一般不能观察到，其主要用在服务器端。JSP只支持非可视化组件。

5.JavaBean范围

在这里面scope说明有不同的范围

  • page 表示只能作用在当前页面，在别的页面中不能认识
  • request 除了可以在当前页面上可用，还可以在通过forward方法跳转页面
  • session 表明可以存在session中，该对象可以被同一个用户的所有页面认识到
  • application 表明可以存在application中，该对象可以被所有用户的所有页面认识到
